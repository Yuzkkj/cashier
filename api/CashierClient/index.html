CashierClient

The client-side of Cashier. Handles purchase prompts, item queries, gift sending, and limited items for the local player.

Types​

ItemType​

</>
type ItemType = "DevProduct" | "Gamepass" | "Semipass"

Determines the type of an item.

Item​

</>
interface Item {
name: string--

The name of the item

description: string--

The description of the item

iconId: string--

The icon of the item. This is a full "rbxassetid://..." string

type: ItemType--

The type of the item

id: number--

The id of the gamepass of developer product linked with the item

giftId: number?--

The id of the developer product for gifting this item

}

Useful information of an item.

Limited​

</>
interface Limited {
name: string--

The name of the limited

id: number--

The developer product id linked to the limited

initialStock: number--

The initial stock when the limited was created

stockLeft: number--

The remaining stock available for purchase

onePerPlayer: boolean--

Determines if a player can only purchase this limited once

expiration: number?--

The Unix timestamp at which the limited expires

}

Configuration and state data for a stock-limited item.

GiftData​

</>
interface GiftData {
item: string--

The name of the item

from: number--

The id of the user that has gifted the item

}

Data from a gift.

Properties​

purchaseStarted​

</>
CashierClient.purchaseStarted: Signal<>

A signal that fires when a purchase prompt is shown to the player.

purchaseFinished​

</>
CashierClient.purchaseFinished: Signal<string,boolean,boolean>

A signal that fires when a purchase prompt is closed, providing the name of the item, a boolean indicating whether the purchase was completed successfully, and a boolean indicating if the purchase was a gift.

limitedChanged​

</>
CashierClient.limitedChanged: Signal<Limited>

A signal that fires whenever a limited's data changes, providing the updated limited info.

Functions​

init​

</>
CashierClient:init() → ()

Initializes the client-side of Cashier.

local Cashier = require(path.to.Cashier).client

Cashier:init()
WARNING

:init(...) should be called before using any other method.

getAllItems​

</>
CashierClient:getAllItems() → {Item}

Returns an array with all registered items.

local Cashier = require(path.to.Cashier).client

for _, item in Cashier:getAllItems() do
    print(item.name, item.type)
end

getItemById​

</>
CashierClient:getItemById(itemId: number) → Item?

Retrieves the info of an item with the given id. itemId can be either a normal id or a giftId. 

local Cashier = require(path.to.Cashier).client

local item = Cashier:getItemById(1234)
for k, v in item do
    print(k, v)
end

getItemByName​

</>
CashierClient:getItemByName(itemName: string) → Item?

Retrieves the info of an item with the given name.

local Cashier = require(path.to.Cashier).client

local item = Cashier:getItemByName("VIP")
for k, v in item do
    print(k, v)
end

getItemsOfType​

</>
CashierClient:getItemsOfType(itemType: ItemType) → {Item}

Returns an array with all items of the given type.

local Cashier = require(path.to.Cashier).client

for _, item in Cashier:getItemsOfType("Gamepass") do
	print(item.name, item.id)
end

isGiftId​

</>
CashierClient:isGiftId(id: number) → boolean

Checks if an id is the giftId of an item.

local Cashier = require(path.to.Cashier).client

local isGiftId = Cashier:isGiftId(4321)
print(isGiftId)

getItemPrice​

This is a yielding function. When called, it will pause the Lua thread that called the function until a result is ready to be returned, without interrupting other scripts. Yields
</>
CashierClient:getItemPrice(itemName: string) → number

Returns the price in robux of an item.

local Cashier = require(path.to.Cashier).client

local priceLabel = script.Parent

local price = Cashier:getItemPrice("VIP")
priceLabel.Text = `${price}`

hasItem​

</>
CashierClient:hasItem(itemName: string) → boolean

Checks if the user has the item with the given name.

local Cashier = require(path.to.Cashier).client

local textLabel = script.Parent

local hasItem = Cashier:hasItem("VIP")
if hasItem then
    textLabel.Text = "Owned"
end

purchaseItem​

</>
CashierClient:purchaseItem(
itemName: string,
...: any?--

Extra params passed to the item's predicate

) → ()

Prompts the local player to purchase the item with the given name.

local Cashier = require(path.to.Cashier).client

local button = script.Parent
button.Activated:Connect(function()
    Cashier:purchaseItem("VIP")
end)

gift​

</>
CashierClient:gift(
userId: number,
itemName: string
) → (
boolean,--

Whether the gift was successfully initiated

string?--

An error message in case the user could not gift the other one

)

Prompts the user to purchase the item for the other player.

local Cashier = require(path.to.Cashier).client

local giftButton = script.Parent
giftButton.Activated:Connect(function()
    Cashier:gift(6969, "VIP")
end)

onGiftReceived​

Override
</>
CashierClient:onGiftReceived(giftData: GiftData) → ()

Overridable method called when the user receives a gift.

local Cashier = require(path.to.Cashier).client

function Cashier:onGiftReceived(giftData: GiftData)
    print(`{giftData.from} gifted you {giftData.item}!`)
end

getAllLimiteds​

</>
CashierClient:getAllLimiteds() → {Limited}

Returns an array with all currently available limited items.

local Cashier = require(path.to.Cashier).client

for _, limited in Cashier:getAllLimiteds() do
	print(limited.name, limited.stockLeft)
end

getLimitedByName​

</>
CashierClient:getLimitedByName(name: string) → Limited?

Retrieves a limited item by its name. Returns nil if no limited with the given name exists.

local Cashier = require(path.to.Cashier).client

local limited = Cashier:getLimitedByName("Exclusive Sword")
if limited then
	print(limited.stockLeft)
end

getLimitedFromId​

</>
CashierClient:getLimitedFromId(id: number) → Limited?

Retrieves a limited item by its developer product id. Returns nil if no limited with the given id exists.

local Cashier = require(path.to.Cashier).client

local limited = Cashier:getLimitedFromId(123456)
if limited then
	print(limited.name, limited.stockLeft)
end

purchaseLimited​

</>
CashierClient:purchaseLimited(limitedName: string) → ()

Prompts the local player to purchase a limited item. Performs client-side validation (stock, expiration, one-per-player) before requesting the server. 

local Cashier = require(path.to.Cashier).client

local button = script.Parent
button.Activated:Connect(function()
	Cashier:purchaseLimited("Exclusive Sword")
end)

hasLimited​

</>
CashierClient:hasLimited(limitedName: string) → boolean

Checks if the local player owns the given limited item.

local Cashier = require(path.to.Cashier).client

if Cashier:hasLimited("Exclusive Sword") then
	print("Player owns the Exclusive Sword!")
end
